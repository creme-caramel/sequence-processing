# Sequence-processing

* Quantifies somatic DNA mutations
* Exports results to db/csv

### To compile

    make clean
    make

### To run

    ./main /path/to/reads_data.fna Mid_number

Eg, `./main /path/to/1.TCA.454Reads.fna 15`<br>

### Dependency

* [mafft](http://mafft.cbrc.jp/alignment/software/)
